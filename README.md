# CMPT 263 Final Exam Trainer

Fast tactical React + Vite + TypeScript study app for SFU CMPT 263: Human-Centered Computing final review.

## Local Development

```bash
npm install
npm run dev
```

Build + lint:

```bash
npm run build
npm run lint
```

## Deployment Note

`vite.config.ts` uses `/cmpt263-exam-trainer/` as the build base path for GitHub Pages style deployment.
